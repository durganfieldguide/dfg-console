# Archived Code

This directory contains legacy code that is no longer in active development.

## Contents

- `frontend/` - Legacy Next.js 16 frontend app. The production app is now at `apps/dfg-app/` which deploys to Vercel as `dfg-console`.

## Why Archived?

These apps were superseded by the consolidated `apps/dfg-app` project which is the single source of truth for the DFG console frontend.

Do not deploy or develop against these archived directories.
